Submit

Be sure to be sure to be a member of tomsk-submit organization on Github.

After that, you can either install the script or use it through docker.

Install

If you opt to install it, should make sure to have python3 installed. Then download the code here, and install it: sudo python3 setup.py install

After that process is done, just run: ptom exerciseName


Docker

Other option is to pull the docker image, and run it:

docker pull barci/ptom:latest
docker run -ti --rm --volume $(pwd):/submit --workdir /submit barci/ptom exerciseName
